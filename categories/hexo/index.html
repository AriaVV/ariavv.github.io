hexo yelee主题头像设置

Jelon 发表于 2017-03-13    标签： 头像设置

前言

最近自己在设置hexo博客主题的时候,关于头像遇到了问题,关于网上的给予的解答并不能够使我的问题解决

主题头像链接设置失败

我在themes-->yelee-->_config.yml文件中,设置了

`avatar: https://github.com/AriaVV/img_user/blob/master/my_avatar.png`

当然结果是失败了,一直显示为:

我在网上搜索资料,试了好几种解决方式

第一种:(失败)
直接修改layout/_partial/left-col.ejs的第六行和第八行为：
<img src="<%=theme.avatar%>" class="js-avatar show">
<img src="<%=theme.avatar%>" class="js-avatar show" style="width: 100%;height: 100%;opacity: 1;">

第二种:(失败)
把头像图片放在themes-->yelee-->source-->img文件夹下,然后在
themes-->yelee-->_config.yml文件中,设置`avatar: /source/img/my_avatar.png`

最后发现,上传到github的文件的img文件夹是不在source文件夹下的,所以直接在 themes–>yelee–>_config.yml文件中,配置
avatar: /img/my_avatar.png就可以完美的显示头像了!!!

【end】

hexo博客部署

Jelon 发表于 2017-03-13    标签： hexo环境、部署、基本操作

前言

最近想要去做一个自己的博客，查了很多的资料，各有各的有点，现用hexo + github部署

配置环境

Node的环境配置(最好安装绿色版本,自己带有npm)

Git的安装(git用于把hexo部署到github)

github账号(申请github账号)

安装全局的hexo

npm install -g hexo

初始化hexo

创建一个放你的博客的文件夹,在当前文件夹打开命令行(shift+右键),输入hexo init

生成静态文件

hexo generate

预览博客

hexo server 

hexo的这个服务器的默认端口号是4000,所以你只需要在浏览器输入http://localhost:4000 就可以预览到你创建的博客了

github配置初仓储

在github上面新建一个你自己的仓储,必须是  `仓储名.github.io`格式

建立关联

在你的本地博客的文件夹下打开_config.yml文件,如下配置:

deploy:
type: git
repository: https://github.com/AriaVV/ariavv.gtihub.io.git
branch: master

博客部署到github

npm install hexo-deployer-git --save

发布(每次更新需要)

hexo clean && hexo g && hexo d

【end】
