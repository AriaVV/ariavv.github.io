Vue交互

Jelon 发表于 2017-03-31    标签：

###引包
Vue本身是不支持交互,如果想要交互,需要引入库 vue-resouce

<script src="./libs/vue.js"></script>
<script src="./libs/vue-resource.js"></script>   //交互

get方式进行交互(开启服务器)

<body>    
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data:{
                  msg:'hello world!....'
              },
              methods:{
                  getData:function(){
                      this.$http.get('get.php',{a:1,b:2}).then(function(res){
                        //成功
                          console.log(res.data);
                      },function(res){
                        失败
                        console.log(res.data);
                      })
                  }
              }
            })
        }
    </script>

    <div id="box">
        <input type="button" value="点击按钮" @click="getData()" />
    </div>
</body>

post方式进行交互(开启服务器)

post方式请求数据必须加 emulateJSON:true

    <body>
      <script>
        window.onload=function(){
            new Vue({
                el:'body',
                data:{

                },
                methods:{
                    getData:function(){
                        this.$http.post('post.php',{
                            a:1,
                            b:2
                        },{
                            emulateJSON:true
                        }).then(function(res){
                            console.log(res.data);
                        },function(res){
                            console.log(res.data);
                        });
                    }
                }
            });
        };
    </script>
    <input type="button" value="按钮" @click="getData()">
</body>

jsonp(跨域)

需要定义callback的指向 jsonp:’cb’ //定义callback的名称,默认是 callback

<body>    
    <script>
        window.onload = function() {
            var app = new Vue({
                el: '#box',
                data: {
                    msg: 'hello world!....'
                },
                methods: {
                    getData: function() {
                        //请求地址:"https://sp0.baidu.com/5a1Fazu8AA54nxGko9WTAnF6hhy/su?wd=a&cb=abc"
                        this.$http.jsonp('https://sp0.baidu.com/5a1Fazu8AA54nxGko9WTAnF6hhy/su', {
                            wd:'a'
                        }, {
                            jsonp:'cb'     //定义callback的名称
                        }).then(function(res) {
                            console.log(res.data.s);
                        }, function(res) {
                            console.log(res.data);
                        });
                    }
                }
            })
        }
    </script>

    <div id="box">
        <input type="button" value="点击按钮" @click="getData()" />
    </div>
</body>

Vue中常用的简单过滤器

Jelon 发表于 2017-03-31    标签：

过滤器用来过滤模板中的数据

常用简单过滤器

语法：{{msg|filterA|filterB}}
传参语法：{{msg|filterA '参数'}}   这个参数就是你想要的参数

1、|uppercase 小写转大写

<body>    
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data: {
                msg:'hello world'
              }
            })
        }
    </script>

    <div id="box">
        {{ msg | uppercase}}
    </div>
</body>

效果图

2、|lowercase 大写转小写

3、|capitalize 首字母大写

4、|currency 美元

5、|currency ‘￥’ 传参，人民币

6、|debounce 2000 延迟执行 一般配合事件使用

<input type="text" @keyup="show() | debounce 2000" />

7、|limitBy 2 取2条数据,跟数据配合来使用

<li v-for="val in arr | linitBy 2"></li>

8、|limitBy 2 1 从索引为1的开始，取2条数据,跟数据配合来使用

9、|filterBy ‘a’ 获取包含’a’的数据

<li v-for="val in arr | filterBy 'a' "></li>

10、|orderBy ‘a’ 排序

<li v-for="val in arr | orderBy 'a' "></li>

11、|orderBy ‘1’ 正序排列

<li v-for="val in arr | orderBy '1' "></li>

12、|orderBy ‘-1’ 倒叙排列

<li v-for="val in arr | orderBy '-1' "></li>

13、自定义过滤器

Vue.filter(名称,function(){})

例子:如果数据小于10,就在前面加上0(自定义一个名称叫做:digit的过滤器)

<body>
    <script>
        window.onload = function() {

            Vue.filter('digit',function(num){
                return num<10?'0'+num:''+num;
            })
            var app = new Vue({
                data: {
                    msg: '7'
                }
            }).$mount('.container');

        };
    </script>
    <div class="container">
        {{msg | digit}}
    </div>
</body>

14、自定义过滤器(时间戳)

<body>
    <script>
        window.onload = function() {
        Vue.filter('time',
            function(date) {
                var date = new Date(date);
                Y = date.getFullYear(),
                    m = date.getMonth() + 1,
                    d = date.getDate(),
                    H = date.getHours(),
                    i = date.getMinutes(),
                    s = date.getSeconds();
                if (m < 10) {
                    m = '0' + m;
                }
                if (d < 10) {
                    d = '0' + d;
                }
                if (H < 10) {
                    H = '0' + H;
                }
                if (i < 10) {
                    i = '0' + i;
                }
                if (s < 10) {
                    s = '0' + s;
                }
                var t = Y + '-' + m + '-' + d +  + H + ':' + i + ':' + s;
                return t;
            });

        var app = new Vue({
            data: {
                msg: Date.now()
            }
        }).$mount('.container');

        };
    </script>
    <div class="container">
        {{msg | time}}
    </div>
</body>

双向数据绑定

当我们使用v-model的时候,就把页面上的所有的msg绑定在了一起
当输入框的msg的值发生改变,页面上其他的msg的值也会发生变化

<body>    
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data: {
                msg:'hello'
              }
            })
        }
    </script>

    <div id="box">
        <input type="text" v-model="msg">
        </br>
        {{ msg }}
        </br>
        {{ msg }}
    </div>
</body>

效果图

只把输入框与第二个绑定,最后一个不变

只需要在第三个中加上* ,如: `{{*msg}}`,这个表示数据只绑定一次

<body>    
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data: {
                msg:'hello'
              }
            })
        }
    </script>

    <div id="box">
        <input type="text" v-model="msg">    //数据更新
        </br>
        {{msg}}                             //数据更新
        </br>
        {{*msg}}                            //数据不更新
    </div>
</body>

效果图

绑定属性

在绑定属性的时候最好用v-bind
写法:<img v-bind:src="url" alt="">
简写:<img :src="url" alt="">

除了class和style所有的属性都可以 :src=”” 这种形式绑定
class在使用时候,可以:class=”[red,blue]” 这里的red是data中定义的数据;
class还可以:class=”{red:true,blue:false}”,这种指的不是data的数据,直接指向类名;
class还可以:class=”json”,json可以在data中定义
style和class的使用方式基本一致

v-for 循环

1、数组语法 (value, index) in arr

value是循环的每一项的值
index是每一项的索引
<body>
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data: {
                arr:['苹果','桃子','菠萝','香蕉']
              }
            })
        }
    </script>

    <div id="box"> 
        <ul>
            <li v-for="(item,$index) in arr">{{ item }}-{{ $index }}</li>
        </ul>
    </div>
</body>

效果图：

如果有重复数据是添加不上的,我们可以通过track-by="$index"添加:
<li v-for="(item,$index) in arr" track-by="$index">{{ item }}-{{ $index }}</li>

2、对象语法 (value, key, index) in object

value是循环的每一项的值
key是每一项的键
index是每一项的索引
<body>
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data: {
                obj:{a:'苹果',b:'桃子',c:'菠萝',d:'香蕉'}
              }
            })
        }
    </script>

    <div id="box"> 
        <ul>
            <li v-for="(item,$key,$index) in obj">{{ item }}-{{ $key }}-{{ $index }}</li>
        </ul>
    </div>
</body>

在对象语法中可以直接使用{{ $key }}表示键,使用{{ $index }}表示索引 `<li v-for="item in obj">{{ $key }}-{{ $index }}</li>`

function in_theaters(){
    var count = 8;       //规定初始的显示数据条数
    var total =8;        //总共的数据条数
    $.ajax({
        url: 'https://api.douban.com/v2/movie/in_theaters',    //豆瓣api
        type: 'GET',
        dataType: 'jsonp',
        data: {
            count:count,
            total:total
        },
        success:function(data){
            //如果地址请求成功把数据渲染在页面上,这里使用artTemplate模板引擎
            $('#in_theaters').html(template('template',{model:data.subjects}));
            //在这里,如果把按钮直接放在页面上,那么如果数据渲染的很慢,那么按钮就会很明显,所以让它和数据一块渲染
            $('.clsDetail').append('<button class="in_more">点击加载更多</button>');
        }
    });
    //在点击按钮的时候加载更多数据,因为按钮是动态渲染到页面的,所以需要用到**事件委托**
    $('.clsDetail').on('click','.in_more',function(){
        //在没有点击按钮之前我们已经渲染了8条数据了,这里让每次点击按钮的时候显示在页面的数据增加8条
        count += 8;  
        //点击按钮的时候禁用按钮,并且继续发起请求获取数据  
        $(this).html('<i></i>正在努力的加载中...').attr('disabled','true').css('backgroundColor','#ccc');
        $.ajax({
            url: 'https://api.douban.com/v2/movie/in_theaters',
            type: 'GET',
            dataType: 'jsonp',
            data: {
                count:count,
                total:total
            },
            success:function(data){
                //获取到更多的数据之后继续在页面渲染数据
                $('#in_theaters').html(template('template',{model:data.subjects}));
                //并且启用按钮的点击功能
                $('.in_more').html('点击加载更多').css('backgroundColor','#1c8bd0').removeAttr('disabled');
                //这里的data.total是我们发起请求后获取到的数据的总条数
                //如果count>data.total,那么就说明没有更多的数据了,我们就给用户一个提示"没有更多数据了",并且把按钮禁用掉
                if(count>data.total){
                    $('.in_more').html('没有更多了').attr('disabled','true').css('backgroundColor','#ccc');
                }
            }
        })
    });
}

scroll-view标签必须要有高度

windowHeight 窗口高度

upper-threshold 距顶部/左边多远时（单位px），触发 scrolltoupper 事件

lower-threshold 距底部/右边多远时（单位px），触发 scrolltolower 事件

bindscrolltoupper 滚动到顶部/左边，会触发 scrolltoupper 事件

bindscrolltolower 滚动到底部/右边，会触发 scrolltolower 事件

var app = getApp()
var start = 0       //豆瓣API的参数
var count = 6       //每次显示的数据条数
var total = 10        //总数据条数,随便给的,因为会动态加载
var apiUrl = 'https://api.douban.com/v2/movie/in_theaters';    // 请求链接
Page({
  data: {
    list: [],                     //根据豆瓣api获取到的数据
    title: 'Loading...',
    loading: true,                //刚开始数据没加载完成时,显示正在加载,提高用户体验
    windowHeight: 0,              //窗口高度
    data_view: 'data_none',       //在用户下拉的时候,如果有数据则隐藏,没有数据了就显示
    data_load: 'data_none',       //在用户下拉的时候,如果有更多数据就显示,没有数据了就隐藏
    data_pull: 'data_none'        //在用户上拉的时候,显示2秒
  },
  onLoad(params) {
    const _this = this
    //获取设备讯息
    wx.getSystemInfo({
      success: function (res) {
        _this.setData({ windowHeight: res.windowHeight })
      }
    })
    wx.request({
      url: apiUrl,
      data: { start: start, count: count, total: total },
      header: {
        'Content-Type': 'json'
      },
      success: function (res) {
        console.log(res.data)
        total = res.data.total
        _this.setData({ list: res.data.subjects, title: res.data.title, loading: false, data_load: 'data_block', data_view: 'data_none', data_pull: 'data_none' })

      }
    })
  },
//上拉刷新
  upper: function (params) {
    console.log('滑动到顶部')
    count = 6
    const _this = this
    wx.request({
      url: apiUrl,
      data: { start: start, count: count, total: total },
      header: {
        'Content-Type': 'json'
      },
      success: function (res) {
        total = res.data.total
        _this.setData({ list: res.data.subjects, title: res.data.title, data_pull: 'data_block' })
        setTimeout(function () {
          _this.setData({ list: res.data.subjects, title: res.data.title, data_pull: 'data_none' })
        }, 2000)
      }
    })
  },
//下拉加载更多
  lower: function (e) {
    console.log('滑动到底部')
    const _this = this
    if (count <= total) {
      count += 6
      wx.request({
        url: apiUrl,
        data: { start: start, count: count, total: total },
        header: {
          'Content-Type': 'json'
        },
        success: function (res) {
          total = res.data.total
          _this.setData({ list: res.data.subjects, title: res.data.title, data_load: 'data_block', data_view: 'data_none' })
          // _this.setData({ list: _this.data.list.concat(res.data.subjects) })
        }
      })
    } else if (count > total) {
      count = total
      console.log(total)
      _this.setData({ data_view: 'data_block', data_load: 'data_none' })
    }
  },
//滑动过程中
  scroll: function (e) {
    console.log('滑动过程中')
  }
})
