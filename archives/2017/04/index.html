jQuery.lazyload.js实现图片懒加载功能(基本使用)

Jelon 发表于 2017-04-20    标签：

引包

jQuery.lazyload插件依赖于jQuery

我们在使用的时候需要引入两个文件

<script src="js/jquery.min.js"></script>

<script src="js/jquery.lazyload.js"></script>

HTML结构

使用这个插件,必须给img标签添加统一的Class

另外src属性是用来占位

data-original属性才是图片的路径

另外图片的高宽是一定要设置的,不然容易出现问题

<img class="lazy" alt="" src="img/example.jpg" data-original="img/example.jpg" />

JS文件

$(window).load(function () {
    $("img.lazy").lazyload({
        placeholder:'images/loadimg.jpg'   //这是占位用的,一般是无用的
    });
});

设置临界点

如果你想提前的加载图片,需要设置threshold选项

如: threshold:200  在距离屏幕200px的位置提前加在图片

$("img.lazy").lazyload({
    threshold : 200
});

设置图片出现的动画效果

$("img.lazy").lazyload({
    effect : "fadeIn"
});

【end】

Vue中的动画

Jelon 发表于 2017-04-05    标签：

最简单的动画

<style>
    .box{width: 100px;height: 100px;background-color:red;}
    /*动画的时间*/
    .fade-transition{transition:all 2s ease;}
    /*进入*/
    .fade-enter{opacity:0;}
    /*离开*/
    .fade-leave{opacity:0;transform:translateX(200px);}
</style>
<body>
    <script>
    window.onload = function() {
        var app = new Vue({
            data: {
                start: false
            },
            methods:{
                toggle(){
                    this.start = !this.start;    //控制.box的显示隐藏
                }
            }
        }).$mount('.container');

    };
    </script>
    <div class="container">
        <input type="button" value="按钮" @click="toggle">
        <div class="box" v-show="start" transition="fade"></div>       //transition="fade"定义动画
    </div>
</body>

利用animate.css定义动画

代码:

<script src="./libs/vue.js"></script>
<link rel="stylesheet" href="./libs/animate.css">
<title>vue动画</title>
</head>
<style>
    .container{width: 100px;margin:100px auto;}
    .box{width: 100px;height: 100px;background-color:red;}
</style>
<body>
    <script>
    window.onload = function() {
        var app = new Vue({
            data: {
                start: false
            },
            methods:{
                toggle(){
                    this.start = !this.start;
                }
            },
            // 定义所有的动画
            transitions:{
                fadeCartoon:{
                    enterClass:'zoomInLeft',  //进入,这个就是animate.css中定义的那些类名
                    leaveClass:'zoomOutRight'  //离开    
                }
            }
        }).$mount('.container');

    };
    </script>
    <div class="container">
        <input type="button" value="按钮" @click="toggle">
        <div v-show="start" class="box animated" transition="fadeCartoon"></div>
    </div>
</body>
