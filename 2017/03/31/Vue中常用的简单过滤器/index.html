Vue中常用的简单过滤器

Vue

过滤器用来过滤模板中的数据

常用简单过滤器

语法：{{msg|filterA|filterB}}
传参语法：{{msg|filterA '参数'}}   这个参数就是你想要的参数

1、|uppercase 小写转大写

<body>    
    <script>
        window.onload = function(){
            var app = new Vue({
              el: '#box',
              data: {
                msg:'hello world'
              }
            })
        }
    </script>

    <div id="box">
        {{ msg | uppercase}}
    </div>
</body>

效果图

2、|lowercase 大写转小写

3、|capitalize 首字母大写

4、|currency 美元

5、|currency ‘￥’ 传参，人民币

6、|debounce 2000 延迟执行 一般配合事件使用

<input type="text" @keyup="show() | debounce 2000" />

7、|limitBy 2 取2条数据,跟数据配合来使用

<li v-for="val in arr | linitBy 2"></li>

8、|limitBy 2 1 从索引为1的开始，取2条数据,跟数据配合来使用

9、|filterBy ‘a’ 获取包含’a’的数据

<li v-for="val in arr | filterBy 'a' "></li>

10、|orderBy ‘a’ 排序

<li v-for="val in arr | orderBy 'a' "></li>

11、|orderBy ‘1’ 正序排列

<li v-for="val in arr | orderBy '1' "></li>

12、|orderBy ‘-1’ 倒叙排列

<li v-for="val in arr | orderBy '-1' "></li>

13、自定义过滤器

Vue.filter(名称,function(){})

例子:如果数据小于10,就在前面加上0(自定义一个名称叫做:digit的过滤器)

<body>
    <script>
        window.onload = function() {

            Vue.filter('digit',function(num){
                return num<10?'0'+num:''+num;
            })
            var app = new Vue({
                data: {
                    msg: '7'
                }
            }).$mount('.container');

        };
    </script>
    <div class="container">
        {{msg | digit}}
    </div>
</body>

14、自定义过滤器(时间戳)

<body>
    <script>
        window.onload = function() {
        Vue.filter('time',
            function(date) {
                var date = new Date(date);
                Y = date.getFullYear(),
                    m = date.getMonth() + 1,
                    d = date.getDate(),
                    H = date.getHours(),
                    i = date.getMinutes(),
                    s = date.getSeconds();
                if (m < 10) {
                    m = '0' + m;
                }
                if (d < 10) {
                    d = '0' + d;
                }
                if (H < 10) {
                    H = '0' + H;
                }
                if (i < 10) {
                    i = '0' + i;
                }
                if (s < 10) {
                    s = '0' + s;
                }
                var t = Y + '-' + m + '-' + d +  + H + ':' + i + ':' + s;
                return t;
            });

        var app = new Vue({
            data: {
                msg: Date.now()
            }
        }).$mount('.container');

        };
    </script>
    <div class="container">
        {{msg | time}}
    </div>
</body>

效果图
