jQuery插件zclip实现复制功能

Jelon 发表于 2017-03-23

前言

最近公司做的一个项目需要实现代码的复制功能,搜索好久发现jQuery有这么一款插件,现在分享一下使用心得

HTML代码

<div class="main_link">
    <input class="text_copy" placeholder="http://www."/>
    <a class="btn_copy" href="javascript:;">复制</a>
</div>

CSS代码

.main_link input {
    outline: none;
    margin-left: 25px;
    vertical-align: middle;
    display: inline-block;
    padding-left: 10px;
    width: 82%;
    height: 30px;
    line-height: 30px;
    font-size: 16px;
    color: #9b9b9b;
    border: 1px solid #cccccc;
}

.main_link .btn_copy {
    display: inline-block;
    width: 75px;
    height: 30px;
    line-height: 30px;
    text-align: center;
    font-size: 16px;
    color: #fff;
    background-color: #de3131;
    -webkit-border-radius: 3px;
    -moz-border-radius: 3px;
    border-radius: 3px;
    float: right;
}

.btn_copy.alcopy {
    background: #ddd;
    box-shadow: 0 3px 0 #aaa;
}

页面效果

引入文件

<script src="js/jquery.min.js"></script>
<script src="js/jquery.zclip.min.js"></script>

JS代码

记得一定要把把ZeroClipboard.swf引入到项目中 

 if($('.main_link').length>0){
    $(".btn_copy").zclip({
        path: "js/ZeroClipboard.swf",
        copy: function() {
            return $(".text_copy").val();
        },
        afterCopy: function() {
            $(this).addClass('alcopy').text('已复制');
        }
    });
}

开启服务器

我发现在使用的时候,有时候会没有效果,需要你把服务器开起来,这样就可以放心的使用了

你只需要在input中输入内容,点击复制按钮,就可以去粘贴了

这里推荐一种静态服务器开启办法:

打开cmd,安装全局的anywhere 
    - npm install --global anywhere
安装好了全局,你就可以使用服务器了
    - anywhere 端口号
