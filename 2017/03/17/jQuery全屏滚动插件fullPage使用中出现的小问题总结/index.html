jQuery全屏滚动插件fullPage使用中出现的小问题总结

Jelon 发表于 2017-03-17

前言

最近在做一个全屏滚动并且带有动画效果的页面,使用的是jQuery的全屏滚动插件–fullPage

在实现功能的时候,出现了一些小问题,现在记录一下解决的方法

fullPage的使用方法

引入文件

<link rel="stylesheet" href="css/jquery.fullPage.css">

<script src="js/jquery.min.js"></script>

<script src="js/jquery-ui.min.js"></script>

<script src="js/jquery.slimscroll.min.js"></script>

<script src="js/jquery.fullPage.min.js"></script>

HTML文件

<div id="fullpage" class="fullpage">
    <div id="part1" class="section">第一屏</div>
    <div id="part2" class="section">第二屏</div>
    <div id="part3" class="section">第三屏</div>
    <div id="part4" class="section">第四屏</div>
</div>

JS文件

$(function(){
    $('#fullpage').fullpage();
});

想要fullPage插件可以达到预期的效果,需要进行配置

这是我自己的配置:

$(function(){
    if( $('#fullpage').length>0){
        $('#fullpage').fullpage({
            resize: true,           //字体是否随着窗口缩放而缩放        
            verticalCentered: false,   //内容是否垂直居中
            scrollOverflow: false,     //内容过多是否显示滚动条
            menu:true,      //绑定菜单
            navigation: true,      //是否显示导航条
            anchors: ['page1', 'page2', 'page3', 'page4'],


            //滚动到某一屏后的回调函数，接收 anchorLink 和 index 两个参数。
            //anchorLink 是锚链接的名称
            //index 是section的索引，从1开始计算   
            afterLoad: function (anchorLink, index) {
                $('.section').removeClass('current');
                setTimeout(function () {
                    $('.section').eq(index -1).addClass('current');
                }, 100);
            },

            //在配置完上面之后发现,首屏的Css3动画没有效果,所以配置了下面就好了
            //这个回调函数只是在生成页面结构的时候调用,用来初始化首屏
            afterRender: function(){
                $('#part1').addClass('current')
            }

        })

    }
})

每一屏中点击按钮跳到下一屏

只需要在每一屏设置一a链接,href属性的值是下一屏的#id值

<div id="part1" class="section">第一屏
    <a href="#part2">跳到下一屏</a>
</div>

<div id="part2" class="section">第三屏
    <a href="#part3">跳到下一屏</a>
</div>

每屏的导航器设置不同的颜色

在设置之前,导航器是有不同的颜色的,如下图:

现在设计图的要求是更改,如下图,并且第一屏和第三屏颜色一样,第二屏和第四屏颜色一样:

配置每个圆点的大小

#fp-nav span,
.fp-slidesNavspan{width: 16px; height: 16px;}

JS文件

那么我们如果修改fullpage.css中的样式,那么每屏的颜色还是一样,所以我用JS来控制了他的颜色

$('#fullpage').fullpage({
            resize: true,
            verticalCentered: false,
            scrollOverflow: false,
            menu:true,
            navigation: true,
            anchors: ['page1', 'page2', 'page3', 'page4'],
            afterLoad: function (anchorLink, index) {
                $('.section').removeClass('current');
                setTimeout(function () {
                    $('.section').eq(index -1).addClass('current');
                }, 0);
                //如果是第一屏和第三瓶配置一种颜色
                if(index == 1 || index == 3){
                    $('#fp-nav span,.fp-slidesNavspan').css('background','#87c2f8');
                    $('#fp-nav li .active span,.fp-slidesNav.active span').css('background','#fff');
                    $('#fp-nav li a').on('mouseenter',function(){
                        $(this).children().css('background','#fff');
                    })
                    $('#fp-nav li a').on('mouseleave',function(){
                        $(this).children().css('background','#87c2f8');
                        $('#fp-nav li .active span').css('background','#fff');
                    })
                    $('#fp-nav span,.fp-slidesNavspan').css('+background','url(../images/y1.jpg)');
                    $('#fp-nav li .active span,.fp-slidesNav.active span').css('+background','url(../images/y2.jpg)');
                //如果是第二屏和第四瓶配置一种颜色
                }else if(index == 2 || index ==4){
                    $('#fp-nav span,.fp-slidesNavspan').css('background','#bfdffc');
                    $('#fp-nav li .active span,.fp-slidesNav.active span').css('background','#47a2f5');
                    $('#fp-nav li a').on('mouseenter',function(){
                        $(this).children().css('background','#47a2f5');
                    })
                    $('#fp-nav li a').on('mouseleave',function(){
                        $(this).children().css('background','#bfdffc');
                        $('#fp-nav li .active span').css('background','#47a2f5');
                    })
                }
            },
            afterRender: function(){
                $('#part1').addClass('current')
            }
        });

这样就配置好了,真的好麻烦,但是我还没有找到简单的方法
