Vue开发环境配置

Jelon 发表于 2017-03-13

一.首先必须安装node

  1. 在安装node时,注意安装绿色版本,因为绿色版包含了npm,这样你就不用再次安装npm

  2. node安装好,你需要去进行配置环境变量,不然可能是会报错的

  3. node和npm的版本不能太低

二.首先需要安装全局的vue

`$ npm install -g vue-cli`

  1. 当然npm在下载vue时,速度可能会非常慢,所以你可以选择安装淘宝的镜像源

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

  2. 安装好cnpm之后,你就可以使用淘宝镜像进行下载安装了

    $ cnpm install -g vue-cli

  3. 安装好之后vue的命令就可以使用了

三.搭建项目结构

  1. vue使用webpack快速搭建结构

    vue init webpack projectName

  2. 在搭建结构的时候,命令行会提示需要你填入的信息,直接写入就好

  1. 进入到当前的文件夹

    cd projectName

  1. 下载好了结构,还不算完成,如果想要这个项目可以跑起来,你就得在当前目录下载他的依赖

    cnpm install 下载依赖(依赖文件在package.json文件中都有体现)

  2. 项目运行

    npm run dev

    • 打开页面如果出现下图就证明结构创建成功了,快开启你的Vue之旅吧

【end】
