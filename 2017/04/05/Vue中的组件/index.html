slot占位

在模板中slot占位,可以显示更多的内容

<template id="temp">
    <h3>111111111</h3>
    <slot name="ul-slot"> 占位 </slot>
</template>

slot占位之后这个ul标签也可以显示在页面中了
<temp>
    <ul slot="ul-slot">
        <li>1</li>
        <li>2</li>
        <li>2</li>
    </ul>
</temp>
